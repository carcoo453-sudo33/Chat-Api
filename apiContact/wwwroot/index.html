Everything You Need

Production-grade features out of the box — no external configuration required

SignalR WebSockets

Real-time bidirectional messaging with automatic reconnect, JWT-based presence, typing indicators, and room broadcasting.

MongoDB + In-Memory

Persistent MongoDB storage with automatic in-memory fallback. Works out of the box — zero external dependencies for development.

JWT Auth + Refresh

Secure Bearer token auth with server-side refresh token rotation. BCrypt password hashing and role-based access control.

File Storage

Local blob storage with UUID filenames, MIME-type validation, 20 MB limit, path-traversal protection, and rate-limited upload.

Two-Tier Cache

Redis (L2 distributed) with IMemoryCache (L1 in-process) fallback. Automatic L1 population from L2. Degrades gracefully without Redis.

Audit Trail

Structured audit log for every auth event, file operation, and admin action. Admin-only query endpoints with full-text search.

Security Hardening

CSP / HSTS / X-Frame-Options headers, rate limiting, DataAnnotation validation on all DTOs, global exception middleware.

CQRS + MediatR

Clean separation of commands and queries via MediatR. Repository pattern + Unit of Work over MongoDB and in-memory stores.

Global Error Handling

Outermost exception middleware catches everything, logs full context, and returns safe 500 JSON — no stack traces leaked to callers.

API Overview

All routes prefixed with /api · WebSocket hub at /hubs/chat

Auth
POST/api/auth/register

Register a new account. Returns JWT + refresh token.

POST/api/auth/login

Login and receive access + refresh token pair.

POST/api/auth/refresh

Silently renew access token via refresh token rotation.

Users
GET/api/users

List all users sorted by display name.

PATCH/api/users/{id}

Partially update a user profile.

POST/api/users/{id}/status

Set online/away/offline — broadcasts via WebSocket.

Rooms
GET/api/rooms

All rooms sorted by latest activity.

POST/api/rooms

Create a Direct / Group / Channel room with auto-slug.

DELETE/api/rooms/{id}/members/{userId}

Remove a member from a room.

Messages
GET/api/messages/room/{roomId}

Paginated history — oldest → newest.

POST/api/messages

Send a message; auto-broadcasts via SignalR.

DELETE/api/messages/{id}

Soft-delete — content replaced, history preserved.

Files
POST/api/files/upload

Upload file (multipart, max 20 MB). Sandboxed storage.

DELETE/api/files/{fileName}

Delete a file — path-traversal safe, rate limited.

Audit
GET/api/audit/recent

Latest N audit events. Admin only.

GET/api/audit/search

Filter by user, action, date range. Admin only.

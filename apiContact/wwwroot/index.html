Everything you need for chat

Built with modern patterns — no compromises on scalability or DX

SignalR WebSockets

Real-time bidirectional communication via SignalR Hub. Automatic fallback to long-polling. Room groups, typing indicators, online presence.

JWT Authentication

Full identity system with register, login, refresh tokens, role-based access, and password hashing via BCrypt. Secure by default.

MongoDB Ready

MongoDB.Driver with typed collections. Runs in-memory when no URI is set — zero configuration needed for local development.

Redis Pub/Sub

StackExchange.Redis pre-wired for horizontal scaling. Broadcast messages across multiple API instances with zero code changes.

Blob File Uploads

Upload images, documents, and media files up to 20 MB. Files auto-attach to messages with direct download URLs.

Full Swagger UI

Every endpoint documented with examples, request/response schemas, and live try-it-out. Custom dark theme included.

API Endpoints at a glance

REST + WebSocket — everything in one place

Auth 6 endpoints
POST/api/auth/register

Register a new account, get JWT tokens

POST/api/auth/login

Login, returns access + refresh token

POST/api/auth/refresh

Exchange refresh token for new access token

GET/api/auth/me

Get current user profile from JWT

POST/api/auth/logout

Revoke refresh token, set offline

POST/api/auth/change-password

Change password, revoke all sessions

Users 6 endpoints
GET/api/users

List all users with online status

GET/api/users/{id}

Get user by ID

POST/api/users

Create a new user account

PATCH/api/users/{id}

Update display name or avatar

POST/api/users/{id}/status

Set online / offline status

DELETE/api/users/{id}

Delete a user account (admin)

Rooms 6 endpoints
GET/api/rooms

List all chat rooms

POST/api/rooms

Create a room (Direct / Group / Channel)

GET/api/rooms/user/{userId}

Get all rooms a user belongs to

GET/api/rooms/{id}

Get single room details

POST/api/rooms/{id}/members

Add a member to a room

DELETE/api/rooms/{id}/members/{uid}

Remove a member from a room

Messages 6 endpoints
GET/api/messages/room/{roomId}

Paginated message history for a room

POST/api/messages

Send a message — broadcasts via WebSocket

PATCH/api/messages/{id}

Edit message content

DELETE/api/messages/{id}

Soft-delete a message

POST/api/messages/{id}/read

Mark a message as read

GET/api/messages/room/{id}/unread

Get unread count for a room

Files 2 endpoints
POST/api/files/upload

Upload file (up to 20 MB). Optionally attach to a room.

DELETE/api/files/{fileName}

Delete an uploaded file by name

Quick connect (JavaScript)
// npm install @microsoft/signalr import * as signalR from "@microsoft/signalr"; const connection = new signalR.HubConnectionBuilder() .withUrl("/hubs/chat?access_token=YOUR_JWT") .withAutomaticReconnect() .build(); connection.on("ReceiveMessage", (msg) => { console.log(`[${msg.senderName}]:`, msg.content); }); await connection.start(); await connection.invoke("JoinRoom", "room_001");
